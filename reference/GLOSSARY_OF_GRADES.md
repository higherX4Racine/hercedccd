# The Common Core of Data uses long names for student grade levels

The Common Core of Data uses long names for student grade levels

## Usage

``` r
GLOSSARY_OF_GRADES
```

## Format

an object of class `tbl_df/tbl/data.frame`, with 19 rows and 2 columns

- Order:

  `<int>` the sort order of the grade label. Kindergarten is 0

- GRADE:

  `<chr>` a long-ish name for the grade, e.g. "Grade 9"
