# The Common Core of Data uses long names for racial/ethnic identities

The Common Core of Data uses long names for racial/ethnic identities

## Usage

``` r
GLOSSARY_OF_OMB_RACES
```

## Format

an object of class `tbl_df/tbl/data.frame`, with 9 rows and 2 columns

- Census Letter:

  `<chr>` Some Census tables use a one-letter suffix, such as "D", to
  indicate that they pertain to a single OMB race

- RACE_ETHNICITY:

  `<chr>` An OMB racial/ethnic identity, such as "Asian American".
