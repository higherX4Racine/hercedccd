# The Common Core of Data tables include specific subtotals

The Common Core of Data tables include specific subtotals

## Usage

``` r
GLOSSARY_OF_TOTAL_INDICATORS
```

## Format

an object of class `tbl_df/tbl/data.frame`, with 5 rows and 5 columns

- By Race:

  `<lgl>` `TRUE` when each subtotal pertains to one OMB race.

- By Sex:

  `<lgl>` `TRUE` when each subtotal pertains to one sex.

- By Grade:

  `<lgl>` `TRUE` when each subtotal pertains to one grade.

- Includes Adults:

  `<lgl>` `TRUE` when each subtotal includes adult students.

- TOTAL_INDICATOR:

  `<chr>` a description of the subtotals' characteristics
