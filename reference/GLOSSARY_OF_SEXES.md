# The Common Core of Data uses only two sexes

The Common Core of Data uses only two sexes

## Usage

``` r
GLOSSARY_OF_SEXES
```

## Format

an object of class `tbl_df/tbl/data.frame`, with 4 rows and 3 columns

- SEX:

  `<chr>` "Female," "Male," or two different versions of "Other."

- Non-Female:

  `<lgl>` `TRUE` if SEX is "Female," `FALSE` otherwise.

- Non-Male:

  `<lgl>` `TRUE` if SEX is "Male," `FALSE` otherwise.
